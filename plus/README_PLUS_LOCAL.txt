Punto Smart OS Plus - paquete completo

Subir todos estos archivos a la raíz del repo.

Para que Drive funcione:
1. Crear OAuth Client ID en Google Cloud.
2. Autorizar el origen:
   https://gonzalorios83-cell.github.io
3. Autorizar la URL:
   https://gonzalorios83-cell.github.io/gonzadrive/
4. Pegar el Client ID en plus-config.js.

Sin Client ID, la web carga pero no sincroniza con Drive.
